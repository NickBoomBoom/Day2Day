# 🎨 课程管理系统 UI 风格展示

这是一个展示多种UI设计风格的课程管理系统静态页面项目。

## 📁 项目结构

```
uiai/
├── index.html              # 风格选择首页
└── styles/                 # 各种风格的课程页面
    ├── modern/            # ✨ 现代简约风格
    ├── dark/              # 🌙 深色模式
    ├── glassmorphism/     # 💎 玻璃态风格
    ├── neumorphism/       # 🎭 新拟态风格
    ├── brutalism/         # 🔲 野兽派风格
    └── gradient/          # 🌈 渐变色彩风格
```

## 🚀 使用方法

1. **打开首页**
   - 在浏览器中打开 `index.html`
   - 或者双击 index.html 文件

2. **选择风格**
   - 在首页上点击任意一个风格卡片
   - 系统会跳转到对应风格的课程管理页面

3. **浏览页面**
   - 每个页面都包含完整的导航栏、课程列表、分类过滤和分页功能
   - 点击左下角的"返回风格选择"可以回到首页

## 🎨 6种UI风格

### 1. ✨ 现代简约 (Modern)
- 清新简洁的设计
- 卡片式布局
- 柔和的阴影效果
- 适合各类教育平台
- **推荐使用**

### 2. 🌙 深色模式 (Dark)
- 深色主题设计
- 紫色霓虹灯效果
- 减少眼部疲劳
- 适合长时间使用和夜间学习
- **热门选择**

### 3. 💎 玻璃态 (Glassmorphism)
- 毛玻璃效果
- 半透明背景配合渐变
- 极具现代感和科技感
- 动态背景动画
- **时尚潮流**

### 4. 🎭 新拟态 (Neumorphism)
- 柔和的阴影创造立体感
- 元素仿佛从背景中浮起
- 触感真实
- **独特设计**

### 5. 🔲 野兽派 (Brutalism)
- 大胆的配色和粗犷的边框
- 强烈视觉冲击
- 高对比度设计
- 适合创意教育平台
- **个性十足**

### 6. 🌈 渐变色彩 (Gradient)
- 丰富的渐变色彩搭配
- 活力四射
- 动态渐变背景
- 适合年轻用户群体
- **充满活力**

## 📋 页面功能

所有风格的页面都包含以下功能:

### 顶部导航栏
- **进行中课程** - 正在学习、暂停课程、收藏课程
- **体验中课程** - 试用课程、免费课程、精选体验
- **已完成课程** - 已完成列表、证书课程、学习报告
- **系统管理** - 用户管理、课程管理、系统设置
- 用户头像和通知按钮

### 分类过滤
- 全部课程、进行中、体验中、已完成
- 编程开发、设计创意、数据分析、人工智能

### 课程卡片
- 8个示例课程
- 课程图片、标题、描述
- 学习进度条
- 课程标签(进行中/体验中/已完成)
- 课时统计

### 分页器
- 上一页/下一页按钮
- 页码选择
- 当前页高亮显示

## 🎯 设计规范

- **版心宽度**: 1200px (max-width)
- **响应式设计**: 支持移动端、平板、桌面
- **交互效果**: Hover状态、过渡动画
- **无障碍**: 遵循WCAG 2.1 AA级标准

## 🛠 技术栈

- 纯 HTML + CSS
- 无需任何框架或依赖
- 可以直接在浏览器中打开
- 所有样式都内嵌在HTML中

## 📱 响应式断点

- **移动端**: < 768px
- **平板**: 768px - 1024px
- **桌面**: > 1024px

## 🎨 自定义颜色

每种风格都在CSS中定义了CSS变量,可以轻松自定义:

```css
:root {
    --primary: #4F46E5;
    --secondary: #10B981;
    --background: #F9FAFB;
    /* ... 更多变量 */
}
```
